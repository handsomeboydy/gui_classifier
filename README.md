# gui_classifier（无人机分图软件）

面向输电线路巡检（如御 3T / 3T 系列等）航拍照片的**自动分图工具**：基于杆塔经纬度台账与照片 EXIF（经纬度/拍摄时间），将照片按**线路 / 杆塔号 / 类别**自动归档输出。

> 当前版本提供 GUI 一键操作，并支持 **手动飞行 / 自主飞行** 两种分图模式。

---

## 功能特性

- **GUI 一键分图**：选择台账库、源照片目录、输出目录、线路名称、距离阈值即可开始。
- **两种模式**
  - **手动飞行（manual）**：按“最近杆塔 + 距离阈值”将照片分到：`精细化` / `红外照片`，并为每基塔生成空的 `通道` 文件夹。
  - **自主飞行（auto）**：先提取 **通道照片**（按拍摄时间窗判定），再对剩余照片进行 `精细化` / `红外照片` 分类（会跳过已进入通道目录的文件，避免混入）。
- **双回路侧别过滤（可选）**：支持从同目录下的 `*双回塔台账文件.xlsx` 读取侧别配置；侧别可写成 `左/右`，也支持分段配置如 `1-15:左,16-20:单,21-25:右`（“单”表示不做侧别过滤）。
- **红外命名清洗（自主飞行通道提取阶段）**：会删除“文件名包含 `_T` 但不包含 `_T_`”的红外违例文件（避免干扰分类）。

## 目录结构（输出示例）

输出根目录 `output_root` 下会生成：

```
output_root/
└── 线路名称/
    ├── 精细化/
    │   └── 001/ 002/ ...
    ├── 红外照片/
    │   └── 001/ 002/ ...
    └── 通道/
        └── 001/ 002/ ...
```

（每个塔号目录内为对应照片拷贝副本。）

## 环境要求

- Windows 10/11（推荐），Python 3.8+
- 依赖库：
  - `pandas`
  - `exifread`
  - `openpyxl`（用于读取 `.xlsx` 台账；缺失时可能导致 Excel 读取报错）
- GUI 使用 `tkinter`（Python 通常自带）

安装依赖：

```bash
pip install pandas exifread openpyxl
```

## 台账与文件命名约定

### 1）线路经纬度台账（必需）

GUI 会在你选择的“台账库文件夹”下，按如下规则寻找台账文件：

- 文件名：`{线路名称}经纬度台账.xlsx`

台账中至少需要包含以下列（列名需一致）：

- `杆塔编号`（建议为纯数字字符串：001/1/…）
- `经度`
- `纬度`

> 自主飞行的“通道照片提取”阶段会按 `线路名称` 列过滤（台账需包含 `线路名称` 列且值与输入一致）。

### 2）双回塔侧别台账（可选）

放在同一目录下，文件名匹配：`*双回塔台账文件.xlsx`。

需要包含列：

- `杆塔1名称` / `杆塔2名称`
- `杆塔1方位` / `杆塔2方位`

`方位` 支持写法：

- 全线固定侧别：`左` 或 `右`
- 分段侧别：`1-15:左,16-20:单,21-25:右`（“单”表示该段不做侧别过滤）

### 3）照片命名约定（影响“通道/红外”识别）

- 自主飞行通道提取与精细化/红外判别依赖文件名模式：
  - `_V_`：可见光精细化
  - `_T_`：红外照片
- 手动飞行模式下，红外判别使用文件名中是否包含 `_T`（更宽松）。

## 使用方法（GUI）

1. 运行 GUI：

```bash
python gui_classifier.py
```

2. 在界面中填写/选择：
   - 台账库文件夹（存放 `{线路名称}经纬度台账.xlsx` 的目录）
   - 源照片文件夹
   - 输出根目录
   - 线路名称（用于拼台账文件名）
   - 距离阈值（米，默认 50）
   - 分图模式：手动飞行 / 自主飞行

3. 点击 **开始分类**，查看日志输出，完成后弹窗提示。

> GUI 会在当前工作目录生成 `gui_classifier_config.json`，用于记忆“台账库路径”。

## 使用方法（命令行）

### 手动飞行

```bash
python classify.py --ledger-file "D:\ledger\某线路经纬度台账.xlsx" ^
  --src-folder "D:\photos\input" ^
  --output-root "D:\photos\output" ^
  --line-name "某线路" ^
  --threshold 50
```

### 自主飞行

建议按 GUI 的执行顺序：

```bash
  --src-folder "D:\photos\input" ^

  --line-name "某线路" ^

  --src-folder "D:\photos\input" ^
  --output-root "D:\photos\output" ^
  --line-name "某线路" ^
  --threshold 50
```

（自主飞行阶段会跳过已被归入 `通道` 的照片文件名，避免重复分类。）

## 常见问题（FAQ）

### 1）Excel 读取报错：`Excel file format cannot be determined...`
- 请确认台账文件后缀为 `.xlsx`
- 安装 `openpyxl`：
  ```bash
  pip install openpyxl
  ```

### 2）提示“无 GPS / 未匹配”
- 工具依赖照片 EXIF GPS（经纬度）；如果照片被二次转发/压缩导致 EXIF 丢失，会被跳过。
- 调整“距离阈值（米）”适当增大（默认 50m）。

### 3）自主飞行未找到通道照片
- 通道提取依赖“精细化照片的拍摄时间窗”，请确保照片 EXIF 时间存在且 `_V_` 命名规范。
